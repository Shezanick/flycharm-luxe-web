# Update FlyCharm Drops

**Every day at 9 PM IST**

## Quick Steps

1. Open `drops-simple.html`
2. Find the `allDeals` array (line ~440)
3. Update deals
4. Save
5. Done

## Deal Template

```javascript
{
    from: "Mumbai",
    route: "Mumbai → Paris",
    destination: "France",
    price: "₹45,000",
```

Keep 10-15 deals live. Mix Economy and Business. Verify prices on Skyscanner first.
